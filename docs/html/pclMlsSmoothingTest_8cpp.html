pclMlsSmoothingTest.cpp consists of 3 unit test cases that test the pclMlsSmoothing class.

TEST(pclMlsSmoothingTest, setRadius) will test the setSearchRadius() and getSearchRadius() method. TEST(pclMlsSmoothingTest, setpclCloud) will test the getInputCloud() and setInputCloud() method. TEST(pclMlsSmoothingTest, MlsFiltering) will test the mlsProcess().
